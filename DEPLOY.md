# 🚀 MergeGuard - Kako deploy-ati

## Opcija 1: Vercel (Preporučujem)

### Korak 1: Idi na Vercel
```
https://vercel.com/new/import/github/Baki39/Mergeguard
```

### Korak 2: Settings
- Framework Preset: **Next.js**
- Build Command: `npm run build`
- Output Directory: `.next`

### Korak 3: Deploy
Klikni **Deploy**!

---

## Opcija 2: Netlify (Ako Vercel ne radi)

```
https://app.netlify.com/start/deploy?repository=https://github.com/Baki39/Mergeguard
```

---

- 🌐 Online URL (npr. mergeguard.vercel.app)
